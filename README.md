# 🐦 Flappy Bird с управлением приседом

Интерактивная игра, где ты управляешь птицей с помощью **приседаний в реальной жизни**.

Положение птицы на экране напрямую зависит от положения твоего таза — без клавиатуры, только движение тела.

---

## 🎮 Как это работает
Результат можно посмотреть здесь - https://drive.google.com/file/d/11F5fje7sZXq_WQlhkahHjgNWiLujks-F/view?usp=sharing

1. Камера захватывает изображение с помощью OpenCV  
2. MediaPipe определяет положение тела (pose detection)  
3. Выполняется калибровка:
   - стоя (верхняя точка)
   - присед (нижняя точка)  
4. Положение таза переводится в значение от `0 до 1`  
5. Это значение управляет высотой птицы  

```
стоишь → птица сверху  
приседаешь → птица снизу
```

---

## 🧠 Идея проекта

Вместо обычных “прыжков” реализовано **непрерывное управление**:

```
позиция таза → нормализация → положение птицы
```

## ⚙️ Калибровка

1. Встань прямо → фиксируется верхнее положение  
2. Сделай глубокий присед → фиксируется нижнее положение  

---

## 🎯 Управление

- Встаёшь → птица поднимается  
- Приседаешь → птица опускается
