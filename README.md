# Squat Controlled Box

## Идея проекта

- камера считывает положение тела через **MediaPipe Pose**;
- берётся средняя высота бёдер (`hip_y`);
- сначала проходит калибровка для двух позиций:
  - стоя прямо;
  - самый низкий присед;
- текущее положение пользователя переводится в коэффициент от `0.0` до `1.0`;
- по этому коэффициенту птица перемещается по вертикали в окне игры.

## Как это работает

1. После запуска появится этап калибровки стоя.
2. Затем будет этап калибровки нижней точки приседа.
3. После калибровки начнётся игра.
4. Чем глубже присед, тем ниже или выше на экране будет птица — в зависимости от логики отображения.

Сейчас в проекте используется простая визуализация: жёлтый квадрат вместо полноценного спрайта птицы.
